# -AI-Python-Code-Reviewer
Developed an intelligent web application that analyzes Python code and identifies potential bugs, coding issues, and style violations using automated static code analysis. The system allows users to paste Python code into a web interface and instantly receive detailed feedback, including warnings, suggestions, and a code quality score. This tool helps developers improve code quality and detect common programming mistakes efficiently.
The application is built using Python, Streamlit, and Pylint, providing a user-friendly interface for real-time code analysis.

Key Features
Real-time Python code analysis
Detection of common programming errors and style issues
Automated code quality scoring
Web-based interface for easy interaction
Instant feedback with bug warnings and suggestions

Technologies Used
Python
Streamlit
Pylint
